# Ronjoo Safaris

Luxury Tanzania Safari Experiences

## About

Ronjoo Safaris offers private safaris, bespoke itineraries, and guided expeditions across Tanzania's most extraordinary wild places.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/joshualotha/ronjoo-safari-experience.git

# Navigate to the project directory
cd ronjoo-safari-experience

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start the development server with hot reload
- `npm run build` - Build the project for production
- `npm run build:dev` - Build the project in development mode
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint to check code quality
- `npm test` - Run tests with Vitest
- `npm run test:watch` - Run tests in watch mode

## Technologies

This project is built with:

- **Vite** - Next generation frontend tooling
- **React** - JavaScript library for building user interfaces
- **TypeScript** - Typed superset of JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn-ui** - Re-usable UI components built with Radix UI and Tailwind CSS
- **React Router** - Declarative routing for React
- **React Hook Form** - Performant forms with easy-to-use validation
- **TanStack Query** - Powerful data synchronization for React

## Deployment

Build the project for production:

```sh
npm run build
```

The production-ready files will be in the `dist/` directory.
